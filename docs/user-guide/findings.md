# Findings guide

Findings aggregate claims, evidence, artifacts, importance, validation state, and reproducibility. Open the Findings view to see promoted findings next to disputed claims. Select an entry to inspect its producing node, evidence IDs, confidence status, and reproduction status.

“Validated” is explicit state produced by a validation event. Agent wording alone never upgrades validation. The seeded run contains six validated and two disputed claims so contradicting evidence remains visible.
