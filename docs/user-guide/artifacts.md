# Artifact guide

Artifacts are semantic records with type, MIME metadata, provenance, integrity metadata, bounded preview information, producing node, and related dataset version. Select an artifact in the galaxy or Artifacts view and choose **Open safe preview**.

The current demo renders structured chart/table and Markdown preview data without executing HTML, JavaScript, notebooks, or generated code. The preview declares whether data is complete, sampled, or truncated. The production prompt's DuckDB-Wasm, Parquet, Arrow, PDF, notebook, and sanitized active-format viewers remain deferred and are not implied by the demo.
