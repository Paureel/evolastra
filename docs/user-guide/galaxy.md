# Galaxy guide

The home system is the root analytical objective. Promoted semantic nodes form systems; artifacts orbit their producing node; findings use square evidence marks; anomalies use irregular marks; agents use paired-sensor probe glyphs. Solid and patterned edges encode relationship type.

Drag to rotate the 3D camera through a complete 360° orbit on either axis, Shift-drag (or middle/right drag) to pan, use the wheel to zoom, and double-click a claimed star to enter its orbital System View. `W`/`S` tilt, `A`/`D` rotate, `Home` resets the camera, and arrow keys cycle through inspectable objects. Search (`Ctrl+K`) is the fastest way to focus a known entity. The Text tree view exposes the same semantic graph without requiring the canvas.

Every generated frontier is a connected minimum-spanning network with local neighbor lanes. A sparse inner approach band places some unclaimed systems close to the command system before the field broadens into the outer frontier. Claimed systems receive a deterministic connected spine and full bridges into that frontier, so every visible system belongs to one traversable graph.

Pause animation affects rendering only. Event ingestion and the live projection continue. Reduced-motion and high-contrast controls are in the lower dock.
